Integration With Testing

It is possible to generate an annotations file by using UI testing framework.For example https://github.com/testingisdocumenting/webtau webtau automatically generates an annotations file in addition to capturing a screenshot.

Presentation Mode

Annotated images automatically participate in presentation mode. Annotations appear one by one and scale to match the zoom level of a presentation.
