Glyphicons

Icons can be included using the inlined code plugin. `:icon: cloud` The result will be appear inline in the current paragraph Glyphicon's list of icons is available here: http://getbootstrap.com/components/#glyphicons http://getbootstrap.com/components/#glyphicons

Bullet Points

Icons can also be used as bullet points replacement: * `:icon: time` Time is important * `:icon: cog` Setup must be easy Time is important Setup must be easy
